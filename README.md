## Мародёрство
_Эта опция поможет Вам в создании лута по заранее заготовленным шаблонам._

###### <p align="center">Интерфейс</p>

<p align="center"><img width="500" height="300" alt="Интерфейс мародёрства" src="https://github.com/user-attachments/assets/506405bf-b634-4de3-a2e4-468cb87965aa"/></p>

При нажатии **CTRL + ЛКМ** по шаблону, он будет скопирован. При нажатии **Alt + ЛКМ** по шаблону, откроется окно просмотра, для стандартных шаблонов, или редактирования, для пользовательских:

###### <p align="center">Стандартный шаблон</p>

<p align="center"><img width="1000" height="500" alt="Окно просмотра шаблона" src="https://github.com/user-attachments/assets/1f9a0708-f2d7-43ca-8edc-53eb8dfe5dcc"/></p>

###### <p align="center">Пользовательский шаблон</p>

<p align="center"><img width="1000" height="550" alt="Окно просмотра шаблона" src="https://github.com/user-attachments/assets/57284357-d903-4892-9e74-5c6e382236ef"/></p>

При нажатии **ЛКМ** по шаблону, будет произведён бросок:

<p align="center"><img width="275" height="300" alt="Бросок по шаблону" src="https://github.com/user-attachments/assets/bc0e9752-a1af-423b-9fef-d7696b6da46c"/></p>

При нажатии в окне чата на:
- Количество найденных денег: персонаж заберёт эти деньги.
- Стоимость всей добычи: персонаж моментально продаст всю добычу за половину стоимости и заберёт эти деньги.
- На название предмета: персонаж заберёт этот предмет.
- Стоимость предмета: персонаж моментально продаст этот предмет за половину стоимости и заберёт эти деньги.

## Генератор
_Эта опция поможет Вам в создании предметов с уникальным набором свойств._

###### <p align="center">Интерфейс</p>

<p align="center"><img width="700" height="150" alt="Интерфейс генератора" src="https://github.com/user-attachments/assets/014b6895-7d9d-42c2-bda0-e4be83e8d9e9"/></p>

При выборе типа предмета, интерфейс отображает подходящие для этого предмета шаблоны.

###### <p align="center">Интерфейс</p>

<p align="center"><img width="700" height="550" alt="Интерфейс генератора" src="https://github.com/user-attachments/assets/d26175c9-3cea-4d48-8b8c-27ce8994d22b"/></p>

При наведении на подходящий шаблон, будет отображено его описание. Оно также включает и список входящих в шаблон достоинств и недостатков и отображает их соотношение.

###### <p align="center">Шаблон</p>

<p align="center"><img width="325" height="500" alt="Шаблон генератора" src="https://github.com/user-attachments/assets/c37d8502-d6b9-411c-8b68-9681d731438a"/></p>

Как только будет выбран предмет и, хотя бы, один шаблон, интерфейс отобразит итоговой список всех свойств, которые будут применены к предмету.
_Для свойств "Красивый" и "Прочный" доступно добавление множественного количество. Чтобы изменить добавляемое число, используйте **"ПКМ"** для увеличения и **"Alt + ПКМ"** для уменьшения числа._

###### <p align="center">Интерфейс</p>

<p align="center"><img width="700" height="575" alt="Интерфейс генератора" src="https://github.com/user-attachments/assets/98090aec-5458-4a21-953f-4989f3dd4ede"/></p>

После нажатия на кнопку **"Создать!"**, Вам будет предложено изменить имя предмета. Как только всё будет сделано, Ваше творение появится в списке предметов!
